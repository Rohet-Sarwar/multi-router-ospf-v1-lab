## **Multi-Router OSPFv1 Lab
##**Objective

Designed and implemented a multi-router network using OSPFv1 to achieve dynamic routing, efficient path selection, and full connectivity between multiple subnets.

## **Topology**

This project simulates a multi-router network using Cisco 2911 routers connected through Gigabit and Serial interfaces.

Multiple LAN networks using 192.7.x.0/24 addressing
Serial links used to simulate WAN connections
Switches used for end device connectivity
End devices include PCs, a laptop, and an NTP server
OSPFv1 is configured on all routers for dynamic route exchange

The topology ensures full communication across all networks using OSPF routing.
